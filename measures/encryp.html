ข้อมูลเชิงเทคนิค
  1. โปรแกรมนี้ เข้ารหัสข้อมูลแบบ block ciphers ด้วยวิธีมาตรฐานการเข้ารหัสขั้นสูง (Advanced Encryption Standrad : AES) ของ U.S. National Institute of Standards and Technology (NIST) ซึ่งได้รับการรับรองมาตรฐาน ISO/IEC 18033-3 เมื่อปี พ.ศ. 2548

  2. รหัสจะเจาะได้ยากขึ้น ถ้ามีความยาวมากขึ้น (จำนวนบิทมากขึ้น) โดยจะมีความยาว 128 บิท 192 บิท หรือ 256 บิท แล้วแต่กรณี ขึ้นอยู่กับความยาวของข้อมูลที่นำมาเข้ารหัส รายละเอียดตาม Github repository (crypto-js)

  3. ปัจจุบัน (พ.ศ. 2567) สำนักงานความมั่นคงปลอดภัยไซเบอร์และโครงสร้างพื้นฐานของสหรัฐอเมริกา (Cybersecurity and Infrastructure Security Agency) ยังคงแนะนำให้หน่วยงานของรัฐทุกแห่งในประเทศ ใช้วิธี AES ในการเข้ารหัสข้อมูลข่าวสารลับ

  4. โปรแกรมนี้ เขียนด้วยภาษา HTML/JavaScript เป็น open source จึงไม่มีการบันทึกข้อมูลที่เข้ารหัสไว้แต่อย่างใด หากผู้เข้ารหัสลืม private key และได้ทำลายข้อมูลต้นฉบับแล้ว จะไม่สามารถกู้ข้อมูลต้นฉบับได้ เว้นแต่จะคาดเดา private key ได้อย่างถูกต้อง

  5. ควรกำหนด private key ให้คาดเดาได้ยาก เช่น การนำ password ที่ตนเองจำได้ง่าย มา hash เป็น private key โดยใช้ secure hash algorithm เพื่อให้ได้ private key ที่คาดเดายาก
