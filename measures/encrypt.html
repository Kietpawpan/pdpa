การเข้ารหัสข้อมูล ตามมาตรฐาน ISO 18033-3

มนตรี เกียรติเผ่าพันธ์

Encrypt เป็น web application สำหรับแปลงข้อมูลเป็นรหัสที่ไม่มีใครรู้ความหมาย (encryption) โดยใช้กุญแจส่วนตัว (private key) ในการเข้ารหัส และต้องใช้กุญแจเดียวกัน เพื่อไขรหัสกลับไปเป็นข้อมูลเดิม (decryption) ทั้งนี้ เพื่อธำรงไว้ซึ่งความลับของข้อมูล โดยเฉพาะข้อมูลอ่อนไหวที่ถูกเก็บรวบรวม ใช้ หรือเปิดเผยผ่านระบบดิจิทัล

เป็นการเข้ารหัสแบบ block ciphers ตามมาตรฐานการเข้ารหัสขั้นสูง (Advanced Encryption Standrad : AES) ของ U.S. National Institute of Standards and Technology (NIST) ซึ่งได้รับการรับรองมาตรฐาน ISO/IEC 18033-3 ในปี พ.ศ. 2548 ทั้งนี้ รหัสจะเจาะได้ยากขึ้น ถ้ามีความยาวมากขึ้น (จำนวนบิทมากขึ้น) โดยรหัสที่ได้จากโปรแกรมนี้ จะมีความจำนวน 128 บิท 192 บิท หรือ 256 บิท อย่างใดอย่างหนึ่ง ขึ้นกับความยาวของ private key ที่กำหนดตอนเข้ารหัส รายละเอียดตาม Github crypto-js repository

ปัจจุบัน (พ.ศ. 2567) หน่วยงานความมั่นคงไซเบอร์และโครงสร้างพื้นฐานของสหรัฐอเมริกา (Cybersecurity and Infrastructure Security Ageny) ยังคงแนะนำให้หน่วยงานในสหรัฐอเมริกา ใช้ AES ในการเข้ารหัสข้อมูลแบบ encryption

อนึ่ง โปรแกรมนี้ เขียนด้วยภาษา HTML/JavaScript เป็น open source ไม่มีการเก็บข้อมูลของท่านแต่อย่างใด หากท่านลืม private key และทำลายข้อมูลต้นฉบับแล้ว จะไม่มีใครกู้ข้อมูลต้นฉบับนั้นได้ เว้นแต่จะคาดเดา private key ของท่านได้อย่างถูกต้อง
